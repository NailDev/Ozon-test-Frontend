# Progress 

Переиспользуемый UI-блок для отображения прогресса в виде кольцевой шкалы. Реализован на чистом JavaScript без использования фреймворков и сторонних библиотек.

**Демо:** https://naildev.github.io/Ozon-test-Frontend/

## Возможности

- Начало дуги — 12 часов, конец движется по часовой стрелке при увеличении значения
- Поддержка трех состояний:
  - Normal — отображение текущего прогресса (0–100%)
  - Animated — вращение блока по часовой стрелке
  - Hidden — скрытие блока со страницы
- Адаптивная верстка под портретную и альбомную ориентацию экрана

## API

| Метод | Описание |
|-------|----------|
| `setValue(number)` | Принимает значение от 0 до 100, устанавливает прогресс |
| `setAnimated(boolean)` | Включает/выключает анимацию вращения |
| `setHidden(boolean)` | Скрывает/показывает блок |

## Использование

```javascript
// Инициализация
const progress = new ProgressCircle('#container');

// Управление
progress.setValue(75);        // 75%
progress.setAnimated(true);   // включить вращение
progress.setHidden(false);    // показать блок
